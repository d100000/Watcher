# Watcher
监视本地运行的软件时间及分析自己的工作状态


## 下载地址
```
[http://api.bobdong.cn/public/static/Win/Watcher守望者.exe](http://api.bobdong.cn/public/static/Win/Watcher守望者.exe)
```

## 功能
 - 最小化自动隐藏
 - 本地数据库分析，不上传数据
 - 精细到秒的监控数据
 - 只会记录当前激活的程序数据，使用数据更加精确
 - 添加发呆判断，更加实时监控你的一举一动
